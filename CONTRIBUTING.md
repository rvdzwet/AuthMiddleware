# Contributing

We love pull requests from everyone. By participating in this project, you agree to abide by the code of conduct.

## Setup
- For the .NET SDK: You need the .NET 10 SDK installed. Run `dotnet test` to run the test suite.
- For the Kong Plugin: You need Kong Gateway or Pongo installed to test the Lua scripts.

## Submitting a PR
1. Fork the repo and create your branch from `main`.
2. Ensure you have added extensive inline XML documentation for any new SDK features.
3. Ensure the test suite passes (`dotnet test`).
4. Issue that pull request!
